=====================================================
Передварительные требования
=====================================================
На компьютере должен быть установлен python версии 3.x.
{
	Ссылка для скачивания: https://www.python.org/downloads/
}
На компьютере должен быть установлен pip.
{ 
	Ссылка для скачивания и инструкция: https://pythonworld.ru/osnovy/pip.html
}
Должна быть скачана библиотека pygame

=====================================================
Инструкции для запуска игры:
=====================================================
Чтобы включить игру наберите в коммандной строке:
	python <dir>start.py
	где <dir> - путь до файла.

=====================================================
Управление:
=====================================================
-----------------------------------------------------
Управление меню:
-----------------------------------------------------
	Для навигации по пунктам меню используйте мышку или стрелки "Вверх" и "Вниз" на клавиатуре.
	Для выбора пункта меню нажмите на него левой кнопкой мыши или нажмите на клавиатуре кнопки "Ввод" или "Пробел" или на цифровой клавиатуре нажмите "Ввод".
	Для выхода или возвращения в меню нажмите на клавиатуре "Escape".
-----------------------------------------------------
Управление в игре:
-----------------------------------------------------
	Для передвижения героя воспользуйтесь срелками на клавиатуре.
	Для поставки бомбы нажмите "Пробел".
	Для паузы нажмите "Escape".
